# Templates Directory

Reusable templates for cold emails, resumes, and other job search materials.

## cold-emails/
Email templates organized by:
- Skill set (backend, frontend, devops, etc.)
- City/Location
- Company type (startup, mid-size, enterprise)

## resume-templates/
Structured resume templates:
- Different formats (ATS-friendly, creative, minimal)
- Industry-specific layouts
- Chronological vs. skills-based formats

## How to Use Templates

1. Select template matching your target
2. Copy and customize with your information
3. Replace placeholders like `[YOUR_NAME]`, `[COMPANY_NAME]`, `[POSITION]`
4. Add specific achievements and context
5. Save and track which template you used

## Adding New Templates

- Follow naming convention: `[target]-[type]-template.txt` or `.md`
- Use clear placeholders for customization
- Include notes on best practices
- Share successful templates with team
