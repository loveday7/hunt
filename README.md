# Job Hunt Tracker 🎯

A collaborative platform to track job applications, opportunities, resumes, and outreach across target cities and companies.

## Features

✅ **Job Opportunity Tracking** - Track companies, positions, and application status  
✅ **Resume Management** - Skill-based and location-based resume versions  
✅ **Cold Email Templates** - Customizable templates by skill, city, and company  
✅ **Collaborative Tracking** - Share opportunities and progress with friends  
✅ **Application Analytics** - Monitor your job search progress  

## Target Cities

- Surat
- Nashik
- Ahmedabad
- And more...

## Target Company Types

- Startups
- Mid-size Companies
- Established Organizations

## Quick Start

1. **Add a Job Application** → Update `applications/applications.csv`
2. **Find Companies** → Check `companies/` directory
3. **Use Email Templates** → See `templates/cold-emails/`
4. **Manage Resumes** → Find in `resumes/`
