# Companies Directory

Organized by company type and location.

## Files

- **startups.json** - Early-stage startups and growth companies
- **mid-size.json** - Established mid-size companies (50-500 employees)
- **Add more** - As needed by location or industry

## Company Information Template

```json
{
  "company_name": "Example Corp",
  "website": "https://example.com",
  "city": "Surat",
  "company_type": "Startup",
  "founded_year": 2022,
  "employee_count": "10-50",
  "tech_stack": ["React", "Node.js", "PostgreSQL"],
  "hiring_roles": ["Frontend Developer", "Backend Developer"],
  "hr_email": "hr@example.com",
  "linkedin_company_url": "https://linkedin.com/company/example",
  "notes": "Known for good culture and growth opportunities"
}
```

## How to Add a Company

1. Open the relevant JSON file (startups.json or mid-size.json)
2. Add a new object with company details
3. Ensure all required fields are filled
4. Save and commit
