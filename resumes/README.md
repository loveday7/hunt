# Resumes Directory

Organized by skill and location for easy access and customization.

## Structure

### skill-based/
Resumes tailored for specific technical skills and roles:
- `backend-resume.pdf` - Backend development focused
- `frontend-resume.pdf` - Frontend development focused
- `fullstack-resume.pdf` - Full-stack focused
- `devops-resume.pdf` - DevOps/Infrastructure focused
- `data-engineer-resume.pdf` - Data engineering focused

### city-based/
Resumes tailored for specific cities and regional preferences:
- `surat-resume.pdf` - Surat-specific version
- `nashik-resume.pdf` - Nashik-specific version
- `ahmedabad-resume.pdf` - Ahmedabad-specific version

## Tips for Resume Management

1. **Keep Multiple Versions** - Maintain different versions for different opportunities
2. **Tailor for City** - Highlight relevant experience for each city
3. **Highlight Relevant Skills** - Match job description keywords
4. **Include Local Context** - Mention willingness to relocate or remote work preferences
5. **Update Regularly** - Keep achievements and skills current

## How to Use

1. Pick the appropriate resume based on the job requirements
2. Review for any last-minute updates
3. Submit with personalized cover letter/email
4. Log in applications.csv which resume was used
